AD Manager Pro

v2.2.2
Enterprise Active Directory Management Platform
Modern web-based AD administration for Windows domain environments. Manage users, groups, computers, and OUs from any browser with complete audit logging and role-based access control.
Built with Python FastAPI + React + Tailwind CSS

Executive Summary

What is AD Manager Pro?

A complete browser-based platform that replaces ADUC with a modern, intuitive web interface accessible from any device.

Key Benefits

  • ✓ Access from any browser on any device
  • ✓ No client software installation required
  • ✓ Complete audit trail of all changes
  • ✓ Role-based access for delegation

Enterprise Features

  • ✓ Bulk operations save hours of work
  • ✓ Real-time reports and analytics
  • ✓ Modern dark-themed responsive UI
  • ✓ Enterprise-grade security

  • • HTTP port 8080

Architecture Flow

Browser → FastAPI Backend → Active Directory (LDAP)

All data stored in local SQLite. No external dependencies required.

Workflow Process

  • 1. Helpdesk submits request with reason
  • 2. Admin sees pending badge
  • 3. Admin reviews details + payload
  • 4. Approve OR Reject with reason
  • 5. Auto-execution in AD on approve
  • 6. Complete audit trail

Getting Started

📥 Step 1: Download

git clone https://github.com/your-repo/ad-manager-pro.git

⚙️ Step 2: Run Installer as Admin

powershell -ExecutionPolicy Bypass -File install.ps1

🔒 Step 3: Setup AD Permissions (on Domain Controller)

powershell -ExecutionPolicy Bypass -File setup_permissions.ps1

🌐 Step 4: Access

http://your-server:8080

Login with your Active Directory credentials

⏱️ Total setup time: 15-20 minutes
